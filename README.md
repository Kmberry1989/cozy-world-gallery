
# Cozy World – Gallery System

Features:
- Multiplayer world
- Land ownership
- Inventory system
- Gallery system (upload + place art)

## Run

ONE COMMAND:
cd /Users/kyleberry/Documents/GitHub/cozy-world-gallery
npm install
npm run dev

CLIENT:
cd client
npm install
npm run dev

SERVER:
cd server
npm install
node index.js
